### 1. Overview

Single select lets the user pick one value from a list of options. The trigger field doubles as a search input when the list is open, which makes the component suitable for both short, fixed option sets and long datasets that require filtering.

Single select is always used inside Form field, which provides label, helper text, validation messaging, and layout orientation (vertical or horizontal). Do not use Single select on its own.

### 2. Usage

#### 2.1 When to use

- The user must choose exactly one value from a predefined set of options.
- The option set is long enough that scanning or filtering is faster than reading every option (typically more than 7 options).
- The selection happens inside a form or filter context where space is constrained.
- The options are dynamic or fetched asynchronously and benefit from in-field search.

#### 2.2 When not to use

- The user needs to select more than one value. Use Multi-select instead.
- The option set has 5 or fewer mutually exclusive options that fit in the layout. Use Segment-buttons instead, which exposes all options without an extra click.
- The user needs to enter a free-text value that is not in a predefined list. Use Text input or Combobox with custom entry.
- The selection is binary. Use Toggle or Checkbox.

#### 2.3 Composition

Single select is an atomic component. It does not include a label, helper text, or validation message on its own. It must always be wrapped in Form field, which owns those elements and the field's overall orientation. See Form field for label, helper text, validation, and orientation rules.

### WCAG criteria

<div class="rz-wcag-item">

**1.4.3 Contrast (Minimum)**
Text, placeholder, icons, and the field border must meet a contrast ratio of at least 4.5:1 against the background in default, hover, and focused states. [Read more](https://www.w3.org/TR/WCAG21/#contrast-minimum)
</div>
<div class="rz-wcag-item">

**1.4.11 Non-text Contrast**
The field border, chevron, and focus indicator must maintain a contrast ratio of at least 3:1 against adjacent colors. This applies because the boundary of the control communicates its interactive nature and its current state. [Read more](https://www.w3.org/TR/WCAG21/#non-text-contrast)
</div>
<div class="rz-wcag-item">

**2.1.1 Keyboard**
All functionality of Single select must be operable through a keyboard interface. This applies because the component is a primary input control and must be usable without a pointing device. [Read more](https://www.w3.org/TR/WCAG21/#keyboard)
</div>
<div class="rz-wcag-item">

**2.4.7 Focus Visible**
The focused state of the field must be visually distinguishable from the default and hover states. This applies because keyboard users rely on a clear focus indicator to track their position in the form. [Read more](https://www.w3.org/TR/WCAG21/#focus-visible)
</div>
<div class="rz-wcag-item">

**3.3.1 Error Identification**
Error messaging is owned by Form field. Single select contributes only the visual error border. See Form field for full error identification compliance.
</div>
<div class="rz-wcag-item">

**4.1.2 Name, Role, Value**
The field must expose its role as a combobox, its current value, its expanded/collapsed state, and the active option to assistive technologies. This applies because the component combines a text input with a popup list, which screen readers must announce correctly. [Read more](https://www.w3.org/TR/WCAG21/#name-role-value)

</div>

### 2. Motion

Motion in Single select signals the relationship between the field and the Option list. The list emerges from the field on open and returns to it on close, so the user understands where the options came from. The transition is short to avoid delaying selection.

#### 2.1 Behavior

<table>
  <thead>
    <tr><th>Action</th><th>Change to</th><th>Animate</th><th>Curve</th><th>Duration</th></tr>
  </thead>
  <tbody>
    <tr><td>Click on field (closed)</td><td>Expanded</td><td>Smart animate</td><td>0.25, 1, 0.5, 1 (Ease Out Cubic)</td><td>240ms</td></tr>
    <tr><td>Click on field (open), Esc, or outside click</td><td>Collapsed</td><td>Smart animate</td><td>0.25, 1, 0.5, 1 (Ease Out Cubic)</td><td>160ms</td></tr>
